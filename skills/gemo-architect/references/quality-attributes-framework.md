# Quality Attributes Framework

Use this reference when architecture choices must be evaluated beyond functional correctness.

## Common Architecture Drivers

- security
- authorization integrity
- reliability
- operability
- observability
- scalability
- performance
- cost
- maintainability

## Questions

- Which quality attributes matter most for this feature?
- Which option best preserves those attributes?
- What new risk does each option introduce?
- Which attributes can be relaxed temporarily and which cannot?

## Output Rule

Do not recommend an architecture option without stating the most relevant quality attributes and the
main tradeoffs against them.
