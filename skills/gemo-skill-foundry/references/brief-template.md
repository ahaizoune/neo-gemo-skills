# Gemo Skill Foundry Brief Template

Use this template when you want to create or upgrade a Gemo skill from a short brief.

## Minimal Brief

- brief: what role or gap the skill should solve
- target lane: optional if you already know it
- target repos or surfaces: optional but helpful
- create or upgrade: optional if already known
- must-have behaviors: optional
- anti-patterns to avoid: optional

## Example: Create

Use `$gemo-skill-foundry` to create a new Gemo skill.

Brief: We need a user-facing skill that can evolve the Gemo skill suite itself, benchmark roles
externally, and keep shared ontology docs current.

Target lane: skill-foundry

## Example: Upgrade

Use `$gemo-skill-foundry` to upgrade an existing Gemo skill.

Brief: `gemo-react-reviewer` needs stronger expectations around App Router route posture, stale
session branches, and auth-boundary regressions in scorecard flows.

Target lane: react-reviewer
